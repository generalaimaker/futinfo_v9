## 📝 주요 기능

### 좋아하는 팀/선수 관리
- 팀, 선수, 리그 프로필 페이지에서 하트 버튼으로 즐겨찾기 추가/제거
- 좌측 사이드바에 즐겨찾기한 팀과 리그 표시
- /favorites 페이지에서 모든 즐겨찾기 관리

### 경기 일정
- 날짜별 경기 일정 확인
- 실시간 경기 필터링
- 리그별 그룹화된 경기 표시

### 검색
- 헤더의 검색바로 팀, 선수 검색
- 자동완성 및 검색 결과 페이지
